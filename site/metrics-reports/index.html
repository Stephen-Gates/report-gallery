    Metrics Reports

    The Metrics report shows aggregate...

    There are four types of reports:

    1. Application Portfolio - What is the question about fitness and criticality?
    2. Project Portfolio - What is the question about risk and value?
    3. Provider Portfolio - What is the question about criticality and quality?
    4. Lifecycle and Age
      • How many Applications are Active or being Phased out each month?
      • How many Applications became Active each year?

    Metrics reports are standard LeanIX reports.
